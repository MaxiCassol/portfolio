Mundo Gym

MundoGym is a platform for the sale of gym conditioning products that comes to connect suppliers with companies that need their products in a clean and intuitive web app oriented to the client and SEO optimization. This platform shares the properties of a social platform, giving users the ability to leave their reviews on published products for a greater benefit in terms of trust and engagement. It uses express, postgreSQL, sequelize and JavaScript. On the front-end, it uses the NextJs library with Redux RTK, CSS modules, and tailwind for styling. These are just a few of the many technologies and features that were used in this project, for more detailed information you can find the project on each of the members' github pages.
